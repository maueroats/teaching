Syllabus: Intermediate

This document details the information specific to Intermediate Computer Science. See the all course syllabus information as well.

Materials

  • Notebook - a lightweight paper notebook dedicated to computer science.
  • USB Drive of any size.
  • Picturing Programs book electronic version - downloaded onto flash drive.
  • DrRacket programming environment - installed at home.
  • WeScheme - Online only replacement for DrRacket. Use if you cannot install software. You have to read special instructions to make it work.
