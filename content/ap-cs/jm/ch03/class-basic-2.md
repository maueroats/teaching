# HobbleBug

* Move, wait, move, wait, repeats

---

# WheezerBug

* First 3 ticks: move forward
* Next 2 ticks: waits
* Repeats

--- 

# LongWheezeBug

* Constructor specifies how many ticks for step, wait

```
Bug b = new LongWheezeBug(5,2);
```
